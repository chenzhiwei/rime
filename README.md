# Rime Input Method Engine

First of all, I love Rime.

## Rime on Linux/MacOS

Install Fcitx5 Rime.

```
mkdir -p ~/.local/share/fcitx5/rime
cp -r * ~/.local/share/fcitx5/rime/
```

## Rime on Windows

```
cp -r * /mnt/c/Users/zhiwei/AppData/Roaming/Rime/
```
